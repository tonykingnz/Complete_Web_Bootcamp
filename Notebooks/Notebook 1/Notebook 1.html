Heading tags "<h1-h6>"


To create a heading, use the tags of heading:


This is a level 1 heading created using the h1 tag


Use the heading tag "<h1>" to create one

The tag start with <h1-h6> and end with </h1-h6>

The tag number varies between 1 and 6. The number 1 "One" is the bigger heading and 6 "Six" is the smaller heading. If you write <h7>, <h8>, <h9>,or any other number bigger than 6 "Six", the heading will be 6 "Six". The limits are 1 as the smaller number compatible and 6 as bigger number compatible
